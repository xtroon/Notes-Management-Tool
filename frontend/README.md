# 🎨 Notes Management Tool - Frontend

This is the frontend client for the Notes Management Tool, built with React, Vite, and Tailwind CSS.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env` file and add the backend URL:
   ```env
   VITE_API_BASE_URL=http://localhost:9000
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

> **Note:** For full project documentation, please see the [Main README](../README.md) in the root directory.
